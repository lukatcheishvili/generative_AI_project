06The Strategist
PFPageForge

06 — The Strategist

It makes the decisions a brand strategist would.

  • Extracts the basics from a free-text brief, then plans.
  • Decides, doesn't write — strategy comes before any copy.
  • Returns strict JSON the rest of the app can trust.
Marketing plan
positioning: "Neighbourhood specialty-coffee bar"
target_customer: "Remote workers & nearby regulars"
value_proposition: "Single-origin coffee, made unhurried"
tone: "Warm, confident, unpretentious"
conversion_goal: "Get people to visit in person"
key_messages: ["Single-origin", "Slow bar", "Local roast"]

Editable

Cheap

11Real-time & Robust

11 — Real-time & Robust

Built to be trusted, not just to demo.

01

Live progress

Streamed st.status updates show each node as it runs (graph.stream, stream_mode="updates").

02

Typed state

A PipelineState TypedDict carries brief → plan → html, validating every node handoff.

03

Keys never leak

Every model call runs in the Python process; keys never reach the browser or the generated file.

04

Guardrails everywhere

Tolerant JSON parsing, friendly errors and real CTA links — no dead or broken pages.
